## 开发环境及 Django 安装

### 虚拟环境配置

使用 `virtualenv` 庄家虚拟环境，用来分离各个工程中不同的包引用。操作系统为 `MacOS Sierra 10.12` 。

{% highlight ruby %}
# 安装 virtualenv
pip install virtualenv

# 创建虚拟环境
virtualenv -p /usr/local/bin/python3.X vlenv

# 导入虚拟环境
source /vlenv/bin/activate

# 查看当前环境下安装包
pip list
{% endhighlight %}

### Django 安装

使用 `pip` 安装 Django 最新版本。

{% highlight ruby %}
# 安装 Django
pip install django
{% endhighlight %}
